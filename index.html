Scoring Algorithm

Each option is scored by looking up census stat deltas in ns_results.csv, combined using weights from priority.cfg.

Higher absolute priority = more weight. Positive = want higher. Negative = want lower. Deltas are normalized by standard deviation.

Formula

for each configured stat:
  norm   = delta / std_dev
  dir    = norm × sign(priority)
  weight = rank_by_abs_priority
  score += weight × dir

→ pick highest total score
