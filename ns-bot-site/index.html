How Scoring Works

The bot scores each issue option by looking up its census stat deltas in ns_results.csv, then combining them using the weights from priority.cfg.

for each configured stat:
  norm   = delta / std_dev(stat)
  dir    = norm × sign(priority_score)
  weight = rank_by_abs_priority
  score += weight × dir

→ pick option with highest total score

Stats with higher absolute priority values get more weight. Positive priority = want the stat to go up. Negative = want it to go down. Deltas are normalized by their standard deviation across all options.
